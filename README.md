# Migrate from a DevExpress DevExtreme React Data Grid to a Bryntum React Grid guide: Starter repository

This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Getting Started

Install the dependencies:

```shell
npm install
```

Populate a local SQLite database with the example data in `src/initialData`:

```shell
node src/addExampleData.js
```

Run the development server using the following command:

```shell
npm run dev
```

This project uses [`next/font`](https://nextjs.org/docs/basic-features/font-optimization) to automatically optimize and load Inter, a custom Google Font.
